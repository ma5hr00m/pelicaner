## 描述

基于 Koa2 和原生 HTML+CSS+JavaScript 实现的网址导航页面，使用 SQLite3 存储数据。

用于杭电助手 2023 级第一次新生分享会的小任务（选做）示范案例，基础较差的新生可仿照此页面完成一个基础的前端小练习，能力强的新生按照自己的路线学习即可。

## 技术栈

- HTML+CSS+JavaScript
- Koa2+SQLite3
